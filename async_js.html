Asynchronous JavaScript

Asynchronous JavaScript allows code execution to continue while waiting for an operation to complete, preventing blocking behavior. This is useful for tasks like fetching data from APIs.

Using Promises

Promises are used to handle asynchronous operations. They can be in one of three states: pending, resolved, or rejected.

let promise = new Promise((resolve, reject) => { setTimeout(() => resolve("Data loaded"), 2000); }); promise.then(data => console.log(data));

Async/Await

Async functions allow you to write promise-based code in a more readable way using the await keyword.

async function fetchData() { let response = await fetch("https://api.example.com/data"); let data = await response.json(); console.log(data); } fetchData();
